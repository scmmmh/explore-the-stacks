Explore the Stacks README
=========================

First download the NLTK stopwords and Punkt tokenizer into the directory you
are running the ExploreTheStacks app in.

Then run the ExploreTheStacks application in the following order:

#. ExploreTheStacks initialise-database
#. ExploreTheStacks load-books
#. ExploreTheStacks load-illustrations
#. ExploreTheStacks filter-books
#. ExploreTheStacks extract-shelfmarks
#. ExploreTheStacks create-shelves
#. ExploreTheStacks index-data
#. ExploreTheStacks create-keywords
